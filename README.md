 Practical Quiz 2

 Project Description
This project is a 3D dining scene created using **HTML** and **A-Frame**.

The scene contains:
- A table
- A minimum of two chairs positioned around the table
- A Milo tin
- A Milk tin
- Two cups
- Environmental decorations for beautification

All objects were created using **A-Frame primitive shapes** without the use of external 3D models, as required for the practical quiz.

Technologies Used
- HTML5
- A-Frame (WebVR Framework)
